## Prerequisites
- Go
- Go Fiber
- MongoDB
- Postman



## Usage
 Clone the repository :
```
$ git clone https://github.com/sesha-2k3/Human-Resource-Management-System.git
```
 To get SUM file :
 ```
$ go mod tidy
```
 To build exe file :
 ```
$ go build main.go
```
 To run main file :
 ```
$ go run main.go
```
Then use Postman to connect with mongodb using url:

http://locolhost:3000/employee
